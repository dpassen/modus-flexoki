# Modus Flexoki

Themes for [GNU Emacs](https://www.gnu.org/software/emacs/) based on the [Flexoki palette](https://stephango.com/flexoki), built on top of [modus-themes](https://protesilaos.com/emacs/modus-themes).

## Installation

```elisp
(use-package modus-flexoki
  :vc (:url "https://github.com/dpassen/modus-flexoki"
       :rev :newest)
  :config
  (load-theme 'modus-flexoki-light :no-confirm))
```
